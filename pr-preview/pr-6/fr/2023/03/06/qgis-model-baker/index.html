QGIS ModelBaker – OPENGIS.ch

6 mars 2023

Description

A l’issue du cours, les participant-e-s connaissent toutes les fonctions de QGIS Model Baker, savent représenter des modèles INTERLIS dans la base de données et importer et exporter des fichiers de transfert. En outre, une introduction au maniement des conteneurs et des jeux de données est donnée et le Live Validator est présenté. Le cours dure une journée et coûte 660 CHF par personne (incl. repas de midi). Un-e enseignant-e pour 8 personnes maximum et deux enseignant-e-s pour un maximum de 14 personnes. Le cours aura lieu à partir de 5 inscriptions. Inscription

Contenu

  • Introduction et installation de Model Baker
  • Modèle INTERLIS et importation de données avec le Wizard
  • Digression « Héritages et représentation physique ».
  • Saisie des données
  • Digression « Conteneurs et Jeux de données »
  • Validation des données et exportation
  • Digression « UsabILIty Hub »
  • Discussion, apports, propositions d’amélioration

Connaissances préalables

Connaissances de base de QGIS et des bases de données (p. ex. notion de « type de données » avec entier/nombre/date/chaîne de caractères/booléen) ainsi que d’INTERLIS. Les débutant-e-s peuvent suivre le cours accéléré INTERLIS (webinaire).

Logiciel

Les composants logiciels suivants sont requis et recommandés pour le cours

  • Installation de QGIS pour Windows, macOS ou Linux https://download.qgis.org. Nous utilisons la version LTR la plus récente.
  • Installation de la base de données PostgreSQL recommandée (version 11 ou plus récente). Nous allons passer en revue les workflows avec PostgreSQL. Si quelqu’un n’a pas installé de base de données PostgreSQL, il est aussi possible de travailler avec GeoPackage.

Informations supplémentaires

Environ deux semaines avant le début du cours, tous les participants recevront un e-mail avec de plus amples informations.
