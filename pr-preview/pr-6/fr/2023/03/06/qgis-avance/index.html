QGIS Avancé – OPENGIS.ch

6 mars 2023

Description

Après avoir suivi cette formation, les participant-e-s sont en mesure de réaliser des projets complexes à l’aide de QGIS. Ils connaîtront les caractéristiques de différents formats de données tels que PostGIS et GeoPackage, seront capables de configurer des flux de travail pour le traitement des géodonnées avec des outils de traitement, de créer des impressions à l’aide de mises en page, d’importer/exporter des données Interlis et d’utiliser des outils de dessin pour la construction. De plus, ils pourront écrire des expressions et connaîtront leurs applications dans QGIS. Le cours dure deux jours (9:00 – 17:00) et coûte 1190 CHF par personne. Ce prix comprend l’inscription, les documents de formation, la certification officielle et deux repas. Aucun ordinateur n’est mis à disposition, les participant-e-s doivent apporter le leur. Un-e enseignant-e pour 8 personnes maximum et deux enseignant-e-s pour un maximum de 14 personnes. Le cours aura lieu à partir de 5 inscriptions.

Inscription

Programme

  • A propos de QGIS
  • Extensions pratiques
  • Intégrer différents formats de données dans le projet (Geopackage, PostGIS, csv, cartes de fond sélectionnées comme WMS, WMTS, tuiles vectorielles, …)
  • Les expressions : structure et exemples d’application
  • Analyses d’attributs
  • Gestion de base de données avec SQL (par ex. views de données PostGIS)
  • Relations et jointures
  • Symbolisation et étiquetage
  • Édition efficace d’attributs (formulaire de saisie, table d’attributs)
  • QGIS mobile : QField
  • Configuration de QGIS, utilisation de profils d’utilisateurs
  • Importation INTERLIS avec l’extension ModelBaker
  • Construire des workflows avec des outils de traitement et une modélisation graphique
  • Numérisation avancée – géométries, topologie
  • Créer une mise en page pour l’impression (entre autres créer un atlas, intégrer une photo)
  • …

Prérequis

Connaissances de base en QGIS, correspondant au cours de base. Aucune connaissance en programmation n’est nécessaire.

Certification

Ce cours est dispensé par une organisation reconnue par QGIS. La certification des participant-e-s est incluse dans le coût du cours.

Logiciel

Installation de QGIS pour Windows, macOS ou Linux https://download.qgis.org

  • Nous utilisons la dernière version LTR disponible.
  • Il n’est pas nécessaire d’installer des plugins à l’avance.

Informations supplémentaires

Environ deux semaines avant le début du cours, tous les participants recevront un e-mail avec de plus amples informations.
