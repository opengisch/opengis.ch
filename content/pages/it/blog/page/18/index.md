## [Creating non-versioned shared libraries for android](<../../../2011/11/23/creating-non-versioned-shared-libraries-for-android/index.html> "Creating non-versioned shared libraries for android")

In the first vesions I used rpl -R -e libqgis_core.so.1.9.90 “libqgis_core.sox00x00x00x00x00x00x00” $APK_LIBS_DIR and similar hacks to remove the version from the libs. But it was rather hacky.
